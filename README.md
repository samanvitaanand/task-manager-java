# task-manager-java

A command-line task management application built using Java and object oriented programming.

## Features
- Create, delete, and update tasks
- Assign priorities and track completion status
- Store and manage tasks using efficient data structures

## Technologies
- Java
- Object-Oriented Programming
- Data Structures (Lists, Maps)

## What I Learned
- Applying OOP concepts such as classes, encapsulation, and inheritance
- Using lists and maps to manage application data
